ip адрес может иметь следующий тип:

  • Зарезервирован - не выдается клиентам, например, это ip роутеров/свичей.
  • Статический - выдается клиенту администратором и не освобождается после завершения сеанса.
  • Динамический - выдается клиенту автоматически в момент подключения и освобождается после завершения сеанса

Вы можете изменять тип у диапазона адресов, при этом, в диапазоне могут быть частично существующие и не существующие ip.
У существующих будет изменен тип, несуществующие будут созданы с заданным типом.
